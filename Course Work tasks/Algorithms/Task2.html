Сортировки

Условие задачи

Написать статический generic класс Sorters<T> параметр T должен реализовывать интерфейс IComparable. В классе должно быть 2 статических метода
  1. void BubbleSort(T[] array);
    Отсортировать массив пузырьковой сортировкой.
  2. void SelectSort(T[] array);
    Отсортировать массив сортировкой выбора.
