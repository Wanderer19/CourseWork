Перестановки

Условие задачи

Написать статический generic класс PermutationsGenerator В классе должен присутствовать публичный статический метод
IEnumerable<IEnumerable<T>> GetPermutations(List<T> elements)
На входе - список элементов. На выходе - список всех возможных перестановок данных элементов. Каждая перестановка представляет собой список.
