Генерация слов

Условие задачи

Написать класс WordsGenerator, конструктор которого принимает алфавит, в котором будут генерироваться слова. В классе должен присутствовать публичный метод
List GetWords(int maxLength);
На входе - длина слов. На выходе - список всех слов заданное длины, которые можно составить из букв данного алфавита.
