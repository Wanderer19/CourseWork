Поиск в массиве

Условие задачи

Написать статический generic класс Searcher<T> параметр T должен реализовывать интерфейс IComparable. В классе должно быть 2 статических метода
  1. int LinearSearch(T[] source, T element)(использовать алгоритм линейного поиска)
    На вход - массив и элемент. Выход - найти индекс первого вхождение данного элемента в массив, если его нет, то вернуть -1.
  2. int BinarySearch(T[] source, T element) (использовать алгоритм бинарного поиска)
    На вход - отсортированный по возрастанию массив и элемент. Выход - найти индекс первого вхождение данного элемента в массив, если его нет, то вернуть -1.
