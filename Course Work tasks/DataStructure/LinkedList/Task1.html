Свой список

Условие задачи

Реализовать Generic класс LinkedList<T>, реализующий интерфейс IEnumerable<T>, поддерживающий такие операции
  1. void EmplaceBack(T value); (Вставить элемент в конец списка)
  2. void EmplaceFront(T value); (Вставить элемент в начало списка)
  3. Clear(); (Очищает список)
  4. T GetFirstItem(); (Вернуть первый элемент списка; если список пуст, то выбросить InvalidOperationException)
  5. T GetLastItem();(Вернуть последний элемент списка; если список пуст, то выбросить InvalidOperationException)
  6. bool IsEmpty(); (Проверка списка на пустоту);
  7. Все методы, которые поддерживаются Enumerable, для этого должнен быть реалиован метод IEnumerator<T> GetEnumerator()

Указания к решению

Для решения понадобится дополнительный generic класс, который представляет элемент списка. Внутри него должны быть 2 поля (значение элемента и ссылка на следующий элемент в стеке)
