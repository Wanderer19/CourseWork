Свой стек

Условие задачи

Реализовать Generic класс Stack<T>, поддерживащий такие операции :
  1. void Push(T value); (Втолкнуть элемент в стек)
  2. T Pop(); (Вытолкнуть элемент с вершины стека; если стек пусть выбросить исключение InvalidOperationException)
  3. T Peek(); (Вернуть элемент с вершины стека, не выталкивая его оттуда; если стек пусть выбросить исключение InvalidOperationException)
  4. Stack<T> Clone(); (Клонировать текущий экземпляр, желательно за О(1))
  5. bool IsEmpty(); (Проверка стека на пустоту);
  6. void Clear(); (Очищает стек);

Указания к решению

Для решения понадобится дополнительный generic класс, который представляет элемент стека. Внутри него должны быть 2 поля (значение элемента и ссылка на следующий элемент в стеке)
