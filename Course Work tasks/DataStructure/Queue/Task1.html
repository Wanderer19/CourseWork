Своя очередь

Условие задачи

Реализовать Generic класс Queue<T>, поддерживащий такие операции :
  1. void Enqueue(T value); (Втолкнуть элемент в очередь)
  2. T Dequeue(); (Вытолкнуть элемент из начала очереди; если очередь пуста выбросить исключение InvalidOperationException)
  3. T Peek(); (Вернуть элемент из начала очереди, не выталкивая его оттуда; если очередь пуста выбросить исключение InvalidOperationException)
  4. bool IsEmpty(); (Проверка очереди на пустоту);

Указания к решению

Для решения понадобится дополнительный generic класс, который представляет элемент очереди. Внутри него должны быть 2 поля (значение элемента и ссылка на следующий элемент в очереди)
