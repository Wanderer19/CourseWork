Свое дерево

Условие задачи

Реализовать Generic класс BinaryTree<T>, у которого должны быть такие публичные методы :
  1. void AddToTree(T value); (Добавить элемент в дерево)
  2. bool Find(T key); (Проверить, есть ли такой элемент в дереве)
  3. IEnumerable<T> InfixTraverse(); (Вернуть все элементы дерева, порядок которых должен соответствовать инфиксному обходу дерева)
  4. IEnumerable<T> PrefixTraverse(); (Вернуть все элементы дерева, порядок которых должен соответствовать префиксному обходу дерева)
  5. IEnumerable<T> PostfixTraverse(); (Вернуть все элементы дерева, порядок которых должен соответствовать постфиксному обходу дерева)

Указания к решению

Для решения понадобится дополнительный generic класс, который представляет элемент стека. Внутри него должны быть 3 поля (значение элемента и ссылки на правого и левого ребенка) Все обходы выполняются при помощи рекурсии.
