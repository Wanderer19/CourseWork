Префиксы и суффиксы

Условие задачи

  1. Написать класс Word, который содержит 1 свойство: string Content;
    2 метода:
    • IEnumerable<string> GetPrefixes(); (Возвращает все префиксы слова)
    • IEnumerable<string> GetSuffixes(); (Возвращает все суффиксы слова)
  2. Написать статический класс Solver, который решает такую задачу:
    На вход список слов, обернутых в класс из пункта 1
    На выход кортеж из 3 слов word1, word2, word3, причем word3 = (префикс word1 или word2 ) и (суффикс word2 или word1), причем word3 должен быть максимальным по длине из всех таких кортежей, удовлетворяющим условию задачи.
    Сигнатура метода должна быть такой:
    Tuple<string, string, string> Solve(Word[] words);
