Обратный индекс

Условие задачи

  1. Написать класс Document, в котором должны быть 2 свойства (или 2 публичных поля)
    • int Id;
    • string Text;
  2. Написать статический класс InvertedIndexBuilder, в котором должен быть один статический метод: Dictionary<string, List<int>> BuildInvertedIndex(Document[] documents)
    На вход подается список документов
    На выход словарь, в котором ключи - слова из текстов всех документов, а значение - список id документов, в котором встречается данное слово, упорядоченный по возрастанию
